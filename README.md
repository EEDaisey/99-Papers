Here are some papers that I have enjoyed.  I still have many more to add.  Gradually adding them as I re-read and play with each of them. -EED

# Programming Languages
* B. Stroustrup. 1978. On Unifying Module Interfaces.
* B. Stroustrup. 1981. Long Return: A Technique for Improving the Efficiency of Inter-Module Communication. 
* B. Stroustrup. 1982. Classes: An Abstract Data Type Facility for the C Language. 
* B. Stroustrup. 1984. Operator Overloading in C++.
* L. V. Rose and B. Stroustrup. 1985. Complex Arithmetic in C++. 
* B. Stroustrup. 1986. An Overview of C++. 
* B. Stroustrup. 1986. What is Object-Oriented Programming? 
* B. Stroustrup. 1988. Parameterized Types for C++.
* B. Stroustrup. 1988. Type-Safe Linkage for C++.
* C. Sanderson. 2010. Armadillo: An Open Source C++ Linear Algebra Library for Fast Prototyping and Computationally Intensive Experiments.
* D. Eddelbuettel and C. Sanderson. 2014. RcppArmadillo: Accelerating R with High-Performance C++ Linear Algebra.
* D. Eddelbuettel and J. J. Balamuta. 2018. Extending R with C++: A Brief Introduction to Rcpp.

# Systems
* Dilley. 1998. Globally Distributed Content Delivery.
* L. Page, S. Brin, R. Motwani, and T. Winograd. 1998. The PageRank Citation Ranking: Bringing Order to the Web. Note: See PageRank Review Paper in 01-Matrix-Theory.
* S. Brin and L. Page. 1998. The Anatomy of a Large-Scale Hypertextual Web Search Engine.
* N. Fraser. 2009. Differential Synchronization.
* Facebook. 2013. Scaling Memcache at Facebook.

* E. F. Codd. 1990. The Relational Model for Database Management.

* M. Baker. 2016. 1,500 scientists lift the lid on reproducibility.
* S. Greeland. 2016. Statistical tests, P values, confidence intervals, and power: a guide to misinterpretations.
* J. Ioannidis. 2018. The Proposal to Lower P Value Thresholds to .005.
* V. Chavez-Demoulin, P. Embrechts, and M. Hofert. 2016. An Extreme Value Approach for Modeling Operational Risk Losses Depending on Covariates.
* S. Negahban, S. Oh, and D. Shah. 2017. Rank Centrality: Ranking from Pairwise Comparisons.
